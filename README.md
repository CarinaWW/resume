# AI電子レジュメ（ポートフォリオ）

本プロジェクトは、職務経歴書と履歴書をもとに、AI活用で作成した1ページ型の電子レジュメです。  
採用担当者が短時間で把握しやすいよう、視認性と情報構造を重視しています。

## 主な機能

- 氏名表示（漢字ルビ）と年齢表示
- プロフィール概要（生年月日、居住地、在留資格、最終学歴、希望勤務地）
- 学歴・職務経歴・業務実績・スキル・資格の整理表示
- 現職在職期間の動的表示（当日基準で `x年xか月満了` を自動算出）
- スクロール時のセクション表示アニメーション
- モバイル対応レイアウト

## 技術スタック

- HTML
- CSS
- JavaScript（IntersectionObserver / 在職期間計算）

## ファイル構成

- `index.html`: ページ本体
- `styles.css`: デザイン、レスポンシブ、タイポグラフィ
- `script.js`: 表示アニメーションと在職期間の自動計算


## 利用方法

1. リポジトリをクローンします。
2. `index.html` をブラウザで開きます。
3. 内容編集は主に `index.html`、見た目調整は `styles.css`、動作調整は `script.js` で行います。

## 開発方針

- 公開版のため、顔写真および詳細住所・詳細連絡先は非掲載
- 英語表記より日本語中心のUI文言を優先
